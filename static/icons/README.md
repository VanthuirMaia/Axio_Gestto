# Ícones PWA - Gestto

## Tamanhos Necessários

Para o PWA funcionar corretamente, você precisa criar os seguintes ícones:

- **72x72** - icon-72x72.png
- **96x96** - icon-96x96.png
- **128x128** - icon-128x128.png
- **144x144** - icon-144x144.png
- **152x152** - icon-152x152.png
- **192x192** - icon-192x192.png (obrigatório)
- **384x384** - icon-384x384.png
- **512x512** - icon-512x512.png (obrigatório)

## Como Gerar os Ícones

### Opção 1: Usar um Gerador Online (Mais Fácil)

1. Acesse: https://www.pwabuilder.com/imageGenerator
2. Faça upload de uma imagem quadrada (mínimo 512x512)
3. Baixe o pacote ZIP com todos os tamanhos
4. Extraia os arquivos nesta pasta

### Opção 2: Usar Photoshop/GIMP

1. Crie uma imagem 512x512 com o logo "GESTTO"
2. Use o gradiente roxo: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
3. Exporte em PNG e redimensione para cada tamanho listado acima

### Opção 3: Usar Canva (Gratuito)

1. Acesse https://www.canva.com
2. Crie um design 512x512
3. Use o logo "GESTTO" com fundo gradiente roxo
4. Baixe como PNG
5. Use https://realfavicongenerator.net/ para gerar todos os tamanhos

## Design Sugerido

**Fundo:** Gradiente roxo (#667eea → #764ba2)
**Texto:** "GESTTO" ou "G" em branco
**Fonte:** Inter (bold/extra-bold)
**Arredondamento:** 20% (opcional, para design mais moderno)

## Ícone Temporário

Por enquanto, você pode usar o ícone SVG incluído (`icon-temp.svg`) até ter os ícones finais.
Para converter SVG em PNG nos tamanhos necessários, use: https://svgtopng.com/
