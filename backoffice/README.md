# Backoffice (God Mode)
Além das correções de bugs e melhorias de segurança, implementamos um sistema completo de gestão administrativa.

## Funcionalidades
- **Hardware Monitoring**: CPU, RAM, Disco.
- **Log Viewer**: Leitor de logs em tempo real.
- **Database Stats**: Contagem de registros.
- **Theme**: Dark Mode exclusivo para admin.

## Arquivos
- `backoffice/` (App Django)
- `templates/backoffice/` (Templates HTML)
